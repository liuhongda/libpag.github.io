接入SDK

libpag接入指南

基本要求

  • 支持android 4.4及以上系统
  • 推荐使用gralde 3.0及以上版本编译

gradle接入

libpag库已经部署到公司内网，所以首先确保能够连接公司内网.

添加gradle依赖

在项目的build.gradle中，添加maven的内网url：http://maven.oa.com/nexus/content/groups/androidbuild

buildscript {
    repositories {
        maven {
            url "http://maven.oa.com/nexus/content/groups/androidbuild/"
        }
    }
}

然后在app的gradle文件app/build.gradle，添加libpag的库依赖.

dependencies {
    //libpag的核心库
    implementation 'com.tencent.tav:libpag:1.3.3'
}

源码接入

Clone 源码

首先将libpag的源码clone到本地，最好跟你的项目工程在同一级目录。

git clone http://git.code.oa.com/MaoBing/libpag.git

配置settings.gradle文件

修改settings.gradle配置文件，将libpag作为project引入.

include ':libpag'
// projectDir设置成libpag相对于主工程的位置
project(':libpag').projectDir = new File(rootDir.getParent()+'/libpag/android/libpag')

配置APP的build.gradle文件

修改App的builde.gradle文件，将libpag作为project引入。

dependencies {
    //libpag project
    implementation project(':libpag')
}

配置完以后，sync一下，再编译。
